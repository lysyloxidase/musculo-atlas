# Hierarchy

The atlas uses ten semantic levels:

1. Body
2. Musculoskeletal system
3. Region or compartment
4. Muscle
5. Fascicle
6. Muscle fiber
7. Myofibril
8. Sarcomere
9. Protein
10. Domain or atomic detail

Phase 1 validates the traversal:

`Body -> Rectus femoris -> Fascicle -> Sarcomere -> Titin -> Ig domain`
